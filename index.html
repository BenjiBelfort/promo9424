Bonjour, afin de nous entrainer à utiliser Git et GitHub en collaboration, je vous propose ce mini side project

  • Créer une page de présentation, voir mon exemple ci-dessous

  • Créer le fichier CSS de ce mini projet
  • etc...

l'objectif étant de nous familiariser avec l'outil de versioning, notamment en mode collaboratif, car nous serons ammenés à l'utiliser pour notre future carrière !
